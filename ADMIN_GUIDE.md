## üìù How to Use the Admin Panel

### Adding a New Project

1. Click **"Gallery Projects"** in the sidebar
2. Click **"New Gallery Project"** button
3. Fill in the fields:
   - **Project Title**: e.g., "Wedding Film: The Smiths"
   - **Category**: Choose from Commercial, Narrative, Events, Music, BTS
   - **Type**: Video or Image
   - **Video/Image File**: 
     - For Vimeo: `https://vimeo.com/YOUR_VIDEO_ID`
     - For YouTube: `https://youtube.com/embed/YOUR_VIDEO_ID`
     - For local file: just the filename
   - **Thumbnail**: Upload an image (optional)
   - **Description**: 1-2 sentences about the project
   - **Tech Stack**: Click "Add item" for each tool (Sony A7III, Premiere Pro, etc.)
   - **Year**: 2024
   - **Featured**: Check to highlight this project
4. Click **"Save"** (saves as draft)
5. Click **"Publish"** to make it live

The site updates automatically in 1-2 minutes!

## üé¨ Uploading Images & Videos

### For Videos:
**Best Practice:** Upload to Vimeo or YouTube first, then use the URL in the admin panel.

**Vimeo Example:**
1. Upload video to Vimeo
2. Get the video ID from URL: `vimeo.com/123456789`
3. In admin, use: `https://vimeo.com/123456789`

### For Images:
Click "Choose an image" ‚Üí Upload from computer

## üìå Important Notes

- **Draft vs Published**: Changes are saved as drafts until you click "Publish"
- **Site Updates**: After publishing, wait 1-2 minutes for GitHub Pages to rebuild
- **Permissions**: Only users with GitHub access to `islayah5/jovon-portfolio` can edit
- **Safety**: CMS creates commits in GitHub - full version history is preserved

## üÜò Troubleshooting

**Can't log in?**
- Make sure you're logged into GitHub
- Check you have access to the repository

**Changes not showing?**
- Did you click "Publish" (not just "Save")?
- Wait 2-3 minutes for GitHub Pages to rebuild
- Hard refresh browser (Cmd+Shift+R on Mac)

**Image won't upload?**
- Keep images under 5MB
- Use JPG or PNG format
- Try a different image
